####P4 Cherry Pickup

In a N x N grid representing a field of cherries, each cell is one of three possible integers.

0 means the cell is empty, so you can pass through;

1 means the cell contains a cherry, that you can pick up and pass through;

-1 means the cell contains a thorn that blocks your way.

Your task is to collect maximum number of cherries possible by following the rules below:

Starting at the position (0, 0) and reaching (N-1, N-1) by moving right or down through valid path cells (cells with value 0 or 1);

After reaching (N-1, N-1), returning to (0, 0) by moving left or up through valid path cells;

When passing through a path cell containing a cherry, you pick it up and the cell becomes an empty cell (0);

If there is no valid path between (0, 0) and (N-1, N-1), then no cherries can be collected.

  • grid is an N by N 2D array, with 1 <= N <= 50.
  • Each grid[i][j] is an integer in the set {-1, 0, 1}.
  • It is guaranteed that grid[0][0] and grid[N-1][N-1] are not -1.

解答：这道题就比较有意思了。普通的动态规划是只考虑如何优化一次路程最大值，而这道题是考虑往返的最大值。需要注意的是往返的最大值并不是两次优化最大值的叠加，一个很简单的例子如下:

在这个例子中单次优化的结果是从左上沿1走到右下，然后穿过两个额外的1回到原点，而实际的最优结果是把最理想的路径拆开成两条路径，最终通过全部的1.

因此我认为解决这个问题的比较容易理解而且写起来又简单的方法是构造一个三维数组，同时记录两条路径得到的最大值。三维数组分别记录两条路径的长度p、第一条路径的终点位置x1和第二条路径的终点位置x2。即第一条路径长度为p，终点位置为（x1,p-x1），第二条路径长度为p，终点位置为（x2,p-x2）。考虑从（x1,p-x1）到（0，0）、再从（0，0）到（x2,p-x2）的最大值。最终结果令两个终点坐标为右下角的点即可。
