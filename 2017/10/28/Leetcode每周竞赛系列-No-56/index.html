Example 1:

Note:

  1. 2 <= len(nums) <= 10000.
  2. 0 <= nums[i] < 1000000.
  3. 1 <= k <= len(nums) * (len(nums) - 1) / 2.

解答：实际上是考察二分法，需要对算法实际运行次数有所掌握，不是对所有距离进行排序而是将所有点排序然后对各个点的距离进行二分搜索，以此来减少实际的时间复杂度。
